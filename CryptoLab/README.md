# كريبتو لاب

كريبتو لاب هو تطبيق سطح مكتب بسيط لتعلم أساسيات التشفير. يستخدم بايثون وCustomTkinter ومكتبة `cryptography` لشرح التشفير وفك التشفير وإدارة المفاتيح وتوليد الهاش بطريقة سهلة.

## أهداف المشروع

- تقديم أدوات تشفير سهلة الاستخدام لطلاب الأمن السيبراني
- واجهة نظيفة وعصرية مع وضع ليلي
- تسميات واضحة وأزرار كبيرة للمستخدمين غير التقنيين
- شرح المفاهيم الأساسية في صفحة التعليم بلغة بسيطة

## الميزات

- صفحة رئيسية مع دليل بدء سريع
- تشفير النص باستخدام Fernet
- فك تشفير النص مع التحقق من المفتاح
- إدارة المفاتيح: توليد، حفظ، وتحميل المفاتيح
- مولد هاش لـ MD5 و SHA-256
- صفحة تعلم تشرح المفاهيم بطريقة مبسطة
- رسائل أخطاء ودية للحالات التالية: حقل فارغ، مفتاح مفقود، نص مشفَّر غير صالح

## شرح الملفات

- `main.py`: الملف الرئيسي الذي ينشئ النافذة، ويبني الشريط الجانبي، ويعرض كل صفحة من صفحات التطبيق.
- `crypto_utils.py`: دوال توليد مفتاح Fernet، وتشفير النص، وفك التشفير.
- `hash_utils.py`: دوال توليد هاش باستخدام MD5 و SHA-256.
- `key_manager.py`: دوال حفظ المفتاح إلى ملف وتحميله من ملف.
- `ui/__init__.py`: ملف تهيئة حزمة واجهة المستخدم.
- `ui/home_page.py`: صفحة البداية التي تعرض وصف التطبيق وأزرار التنقل السريعة.
- `ui/encrypt_page.py`: صفحة التشفير مع حقل إدخال النص وزر التشفير وزر النسخ.
- `ui/decrypt_page.py`: صفحة فك التشفير مع حقل لصق النص المشفر وزر فك التشفير وزر النسخ.
- `ui/hash_page.py`: صفحة توليد الهاش واختر خوارزمية MD5 أو SHA-256.
- `ui/keys_page.py`: صفحة إدارة المفاتيح لتوليد، حفظ، تحميل، ونسخ مفتاح Fernet.
- `ui/learn_page.py`: صفحة الشرح التعليمي للمفاهيم الأساسية.

## التثبيت

1. ثبّت بايثون 3.
2. أنشئ بيئة افتراضية (موصى به):

```bash
python3 -m venv venv
source venv/bin/activate
```

3. ثبّت الاعتماديات:

```bash
pip install -r requirements.txt
```

## تشغيل كريبتو لاب

من مجلد المشروع، نفّذ:

```bash
python main.py
```

## كيفية الاستخدام

1. ابدأ من الصفحة **الرئيسية**.
2. انتقل إلى **المفاتيح** لتوليد أو تحميل مفتاح Fernet.
3. استخدم **تشفير** لكتابة نص والحصول على نتيجة مشفَّرة.
4. استخدم **فك التشفير** للصق النص المشفَّر واستعادة الرسالة الأصلية.
5. استخدم **هاش** لتوليد MD5 أو SHA-256.
6. زر صفحة **تعلم** لقراءة شرح مبسَّط للمفاهيم.

## ملاحظات للطلاب

- هذا المشروع للتعلم الدفاعي في الأمن السيبراني فقط.
- Fernet يسهل استخدام التشفير المتماثل بطريقة أكثر أماناً.
- الهاش هو عملية اتجاه واحد تساعد في التحقق من سلامة البيانات.
- SHA-256 أقوى من MD5 من حيث الأمان.
